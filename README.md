Descripción del Proyecto
Este proyecto consiste en el desarrollo de una aplicación web que simula un sistema de inicio de sesión y gestión básica de usuarios, utilizando los lenguajes HTML, CSS y JavaScript. El sistema valida el acceso mediante credenciales, controla los intentos fallidos permitiendo un máximo de tres, y bloquea el acceso en caso de superar dicho límite. Una vez autenticado correctamente, el usuario accede a un módulo de administración que permite crear, visualizar, editar y eliminar usuarios, aplicando manipulación del DOM y manejo de eventos sin recargar la página. La información se almacena de forma local a través de localStorage, garantizando la persistencia de los datos. El proyecto fue desarrollado aplicando buenas prácticas de programación y control de versiones mediante Git y GitHub, trabajando de forma colaborativa a través de ramas y procesos de merge.

Objetivo del Proyecto
Aplicar los conceptos fundamentales de desarrollo Frontend mediante la construcción de una aplicación web funcional que evidencie el uso de lógica en JavaScript, interacción con el usuario, manipulación dinámica del DOM y trabajo en equipo con control de versiones.

Conceptos Aplicados
-Variables
-Condicionales
-Funciones
-Manejo de eventos
-Manipulación del DOM
-Almacenamiento local (localStorage)


Instrucciones de Ejecución
Para ejecutar el proyecto, abrir el archivo index.html en un navegador web. El sistema solicitará el ingreso de usuario y contraseña para acceder a la aplicación.
