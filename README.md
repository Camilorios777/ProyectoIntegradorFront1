PROYECTO INTEGRADOR FRONTEND

Sistema de Notas

Descripción: 
Este proyecto simula un proceso de inicio de sesión utilizando el lenguaje de JavaScript, se aplicaron los siguientes conceptos:
- Variables
- Condicionales
- Ciclo while
- Función

Cómo ejecutarlo:
Abrir el archivo index.html en el navegador, luego ingresar el usuario y contraseña cuando el sistema lo solicite.

Las credenciales de prueba para que funcione son las siguientes:
- Usuario: admincesde
- Contraseña: 12345

El Logueo permite máximo 3 intentos, después del tercer intento que no se ingrese correctamente la contraseña o usuario, la cuenta será bloqueada.
